# 关于偏见与傲慢

虽然我自认为不是一个喜欢贴标签的人，但实际上有一样跟贴标签一样令人不适的事情，那就是很容易基于个人所谓的荣誉感，个人价值（可能不太恰当，并不完全这样）而对某人产生一些偏见，而这种偏见还会基于上述的标准而变动（增加或减少），往往在某种感情的影响下而选择性的忽略一些客观的因素，选择性的无视他人的友善，而把自己的想法（可能是对他人不好的揣测）强加于他人，这时往往容易就会误解别人的一些指向性不明显的举动，这样其实很容易会把自己放在和他人对立的一种状态。即使实锤他人的缺点其实并不会给你带来真正的内心平静和舒适，不去聚焦于他人的“缺点”，或者退一步去同样地适当地放大被偏见对象的闪光点，也许就能理解你所看到“偏见”并且释怀。

本来绝大部分情况下你就不需要去实锤别人的缺点，他人以及他人的环境并非是由某人单一来评价观测而形成，每个人的价值都是因周围环境的“观测”而存在，落实到个人的体验还取决于内心的认可。所以努力做好自己就够了，放下无谓的偏执，去追求真正的内心，默然回首，就会发现结束内心世界的战争其实没有那么难。
