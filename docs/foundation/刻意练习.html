# 刻意练习

# 一万小时定律？

  1. 不同领域的技能习得时间并不相同，不能一概而论。
  2. 成功和时间并不完全成正比，且天赋不起决定性作用
  3. 除了时间，还取决于练习的方法

一粒沙子不是堆，两粒也不是，三粒也不是，那么9999粒也同样不是，所以1万小时的边界并不存在

# 隐性知识

认知复杂性高(销售)和认知复杂性低(象棋)的学习差异很大程度上表现为隐性知识的多少和比重。

  1. 隐性知识需要在情境中学习寻找
  2. 成年人最佳的学习方式并非独自练习，而是在情景中学习

# 天才是训练的产物

研究大脑的人员发现（无论哪个年龄段的）大脑比我们想象中的更具有适应能力，莫扎特（还有很多例子）的大脑拥有完美音高的回路，是训练的产物，而非某些天生的基因。天才与我们的差别，可能是更多地利用了这一能力（大脑的适应能力）

往往家庭条件越好，父母越能抽出精力时间和金钱去教育孩子，则他们越早能利用大脑适应能力，塑造出老鼠的儿子会打洞的现象。其实也是属于后天的影响，而非天生如此。大体上，人生而平等。

  1. 学习不是挖掘潜力的方式，可以发展这种潜力的方式，我们可以通过学习创造潜力。
  2. 这种普遍有效提高水平的学习方法即为刻意练习。

如我们将会看到的那样，天生的特征比许多人通常以为的，发挥的作用不但小得多，而且也有很大的不同。

# 学习新技能的常见路线

  1. 学习任何一种技能，通过一些知道慢慢达到可以接受的水平，并且可以自动化。
  2. 往后不再走出舒适圈，并满足于当前的练习，几时再经过长时间练习，也很难有突破。
  3. “天真的练习”：基本只是反复做某件事，并指望只靠这种反复就能提高水平和表现。

# 刻意练习

  1. 走出舒适圈，困难即突破，走出舒适圈但又不至于偏离太远，以致于身心受伤，就像健身，你不可能一天做1w个俯卧撑去快速提升，这样只会由于短期压力而造成肌肉劳损被迫中断，更不利于长周期的执行。要充分利用大脑的适应能力。
    1. 大脑实际上有无效的适应能力。
    2. 大脑对于处于舒适区之外却离得不太远的“甜点”上的挑战，改变最为迅速。
  2. 遇到瓶颈怎么办？
    1. 遇到障碍，最好的方法是从不同方向去尝试突破
    2. 可能是动力不足——倦乏
      1. 有意义的正面反馈是保持动机的关键之一，外在成就感
      2. 内在的驱动力，不是为了赢得比赛，而是肉眼可见的突破自己，比昨天的自己更好，更强。
      3. 内驱力往往比外单纯战胜别人更好，别人可能一开始对于自己来说高耸入云，也有利于个体良性竞争。
    3. 适当逼迫一下自己，譬如更短的时间。
  3. 注意技巧
    1. 对于记忆力，有意义的记忆更重要，即常说的尽量不要死记硬背，国际象棋大师下盲棋不是因为培养了不可思议的记忆力，相反他们的记忆取决于场景，即记住的不是棋子在那个位置，而是为什么会在这个位置-----心理表征
    2. 建立心理表征，有利于纵观全局，而又不失“局点”。

      建立心理表征有点类似于构建体系，明白知乎所以然，关联起来。而且熟能生巧。

    3. 在较短时间内投入百分百的努力来练习效果比更长时间只投入70%的努力效果更好。
    4. 站在巨人的肩膀上：通过导师的指导，或者学习观察模拟领域内杰出人物的优点。
    5. 关注每次可以改良的地方，并且纠正而非一味地重复，建立反馈（学生时代的错题集）
  4. 量变引起质变

    杰出人物的进展只在局外人看来才是重大进展，因为那些人并没有见证过每次那些微小的进展，而正是这些小小的进展，才累积成重大的飞跃。
